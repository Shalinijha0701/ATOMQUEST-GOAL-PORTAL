# AtomQuest Goal Setting & Tracking Portal

Full-stack implementation for the AtomQuest Hackathon 1.0 BRD: employee goal creation, L1 approval, shared KPIs, quarterly achievement updates, manager check-ins, admin governance, audit trail, reporting, and analytics.

## Quick Start

```bash
npm install
npm run dev
```

Client: `http://localhost:3000`  
API: `http://localhost:5000/api`

Copy `.env.example` to `.env` when you want custom settings. The app defaults to `DEMO_MODE=true` so judges can exercise all workflows without waiting for calendar windows. Set `ENFORCE_WINDOWS=true` and `DEMO_MODE=false` to enforce the BRD quarterly dates strictly.

## Vercel Deployment

This project is now configured for Vercel static deployment of the React client with a serverless `/api` backend.

- The static site is built from `client/package.json`.
- API requests under `/api/*` are routed to a Vercel Node serverless handler in `api/[...slug].js`.
- On Vercel, backend data is stored in temporary runtime storage (`/tmp`) and is intended for demo purposes.

To deploy:

1. Push the repo to GitHub.
2. Create a new Vercel project and connect it to this repository.
3. In Vercel settings, no root change is needed because `vercel.json` already points to the `client` build.
4. Optionally add environment variables via Vercel if you need custom settings.

## BRD Coverage

- Phase 1 goal sheets with thrust area, title, description, UoM, target, and weightage.
- Validation: total weightage must equal 100%, minimum goal weightage is 10%, maximum 8 goals.
- L1 manager workflow with inline edits, return for rework, approval, and locking.
- Shared departmental KPIs pushed by manager/admin, with read-only title/target and recipient weightage adjustment.
- Quarterly achievement updates with status and computed progress scores.
- Manager check-in comments and completion dashboard.
- Admin cycle management, unlock exceptions, audit trail, and escalation run.
- CSV achievement report export.
- Analytics for trends, distribution, and manager effectiveness.
